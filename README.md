# @artdeco/website

`@artdeco/website` is a new Node.js npm package.

Components, scrips and styles for the art deco website.

## `artdecoWebsite():void`

Call this function to get a result you want.

```js
const artdecoWebsite = require('@artdeco/website')

artdecoWebsite()
```
